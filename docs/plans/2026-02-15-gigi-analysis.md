# Sintra AI Gigi 能力分析

> 数据来源：sintra.ai/gigi 官网、help.sintra.ai 帮助中心、Lindy/Gmelius 第三方评测

## 定位

Gigi 定位为 **Personal Growth Coach + Virtual Secretary**，核心理念是"减少认知负荷"——用户不需要打开多个 App，一个 AI 助手把日程、任务、目标、习惯、信息整理全部搞定。

**定价：** $39/月

**官方 tagline：** "From daily tasks to long-term planning, Gigi helps you stay organized, focused, and in control."

## 六大能力域

### 1. 日程与时间管理

- 根据用户精力水平生成日/周计划（时间块 time-blocking）
- 创建/读取 Google Calendar 事件
- 跨时区安排会议
- 提醒截止日、约会、跟进事项
- 示例 prompt："帮我规划一个高效但放松的一天"

### 2. 任务管理

- 把零散待办转成优先级排序的任务清单
- 拆解复杂任务为可执行步骤
- 创建重复性任务模板和 checklist
- 跟踪多个并行项目的进度
- 宣称能同时管理 10,000 个任务

### 3. 目标与习惯养成

- 设定目标 → 拆成里程碑 → 定截止日 → 定期问责
- 健身目标（接入 Strava 运动数据）
- 饮食计划（定制膳食方案 + 购物清单）
- 学习规划（考试备考、技能学习时间表）
- 财务目标、生活习惯
- 定期发动力消息和进度可视化

### 4. 信息处理与总结

- 总结笔记、文档、长文内容
- 上传文件/URL/文本，提取关键信息
- 整理数字杂乱（收藏夹、笔记、截图）
- 会议转录 + 生成要点 + 追踪决策 + 跟进邮件

### 5. 沟通辅助

- 起草邮件和私人消息
- 收发 Gmail/Outlook 邮件
- 结构化决策分析（pros/cons）
- 回答团队查询、处理问题

### 6. 记忆与认知减负

- Brain AI 知识库存储用户个人资料、偏好、文档
- 记住用户承诺和习惯
- 跨会话保持上下文（通过 Brain AI）
- 每日摘要推送（如邮件日报）

## 集成生态

| --------------- | -------------------------- |
| Google Calendar | 创建/读取事件              |
| Gmail           | 发/回复邮件、读收件箱      |
| Google Drive    | 保存消息和文档             |
| Strava          | 读取运动数据，优化健身计划 |
| Notion          | 创建/更新页面、读取数据库  |
| Microsoft       | 发/回复邮件、管理文档      |

## 工作流模式

Gigi 不是被动问答，而是有**结构化引导流程**：

1. **Onboarding 阶段** — 问你怎么工作、关心什么、想实现什么目标
2. **Context 注入** — 上传笔记/清单/日程到 Brain AI
3. **优先级设定** — 设定目标、边界、偏好（什么该推、什么该暂停、什么忽略）
4. **执行循环** — Gigi 建议 → 用户审批 → 执行 → 复盘

## 实际局限（第三方评测发现）

| 局限              | 说明                                                   |
| ----------------- | ------------------------------------------------------ |
| **纯对话式**      | 没有真正的自动化，不能自动执行任务，需要用户审批每一步 |
| **助手间隔离**    | Gigi 和其他 Helper 不共享上下文，切换时要手动搬运信息  |
| **不可定制**      | 不能修改 Helper 的行为、不能创建自定义 Helper          |
| **浅执行**        | 评测称"simple, structured tasks"还行，复杂工作流不够深 |
| **无主动推送**    | 没有定时检查/主动提醒机制，需要用户主动来问            |
| **Power-Up 过多** | 90+ 个微工具，缺乏引导，新用户容易迷失                 |

## 核心洞察

Gigi 的核心价值不在工具深度，而在**结构化的生活管理框架** — 把散乱的目标/任务/日程/习惯，通过引导式对话组织成可执行的计划，再定期复盘调整。
