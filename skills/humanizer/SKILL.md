# Humanizer: 去AI味优化

当用户发来一段文本要求"去AI味"、"润色"、"改写得更自然"时，按以下流程处理。

## 处理流程

1. **检测 AI 模式**：扫描文本，识别下方 24 种 AI 写作模式中出现的问题
2. **重写优化**：按 5 条核心原则重写，消除所有检测到的 AI 痕迹
3. **评分**：给出爆款 6 维度 + 人味 5 维度评分
4. **输出**：先给优化后的完整文本，再附评分和改动说明

## 5 条核心原则

1. **删除填充短语和开场白** — 直接进入正题
2. **打破公式结构** — 避免二元对比、三段式排列
3. **变化句子长度和节奏** — 长短交错，避免机械重复
4. **直接陈述，信任读者** — 不过度解释，不说教
5. **删除金句** — 可引用的漂亮句子往往是 AI 痕迹最重的地方，重写掉

## 24 种 AI 模式检测清单

### 内容模式（1-6）

1. **过度强调意义/遗产** — 删除夸大象征，用具体事实替代
2. **模糊知名度声明** — "广受关注""引发热议"→用具体数据或删除
3. **-ing 肤浅分析** — 删除分词短语堆砌
4. **宣传广告语言** — "颠覆性""划时代"→用平实描述替代
5. **模糊归因** — "有专家指出""据了解"→提供具体来源或删除
6. **公式化"挑战与展望"** — 用具体数据和观点替代套话

### 语言模式（7-12）

7. **AI 高频词汇** — 此外、至关重要、深入探讨、赋能、格局、底层逻辑、打通闭环→换成简单词
8. **系动词回避** — 不要刻意避免"是"和"有"，该用就用
9. **否定式排比** — "不是A，不是B，而是C"→直接说C
10. **三段式过度使用** — 总是三项？改两项或四项
11. **同义词循环** — 刻意换词显得不自然，允许适当重复
12. **虚假范围** — "从A到B"的假概括→删除

### 风格模式（13-16）

13. **破折号过度** — 用逗号或句号替代
14. **粗体过度** — 删除不必要的加粗
15. **内联标题列表** — 改为连续段落叙述
16. **装饰性表情符号** — 删除无意义的 emoji

### 交流模式（17-24）

17. **协作痕迹** — 删除"希望对您有帮助""如有疑问请随时提问"
18. **知识截止声明** — 删除
19. **谄媚语气** — "这是一个非常好的问题"→用中性语言
20. **填充短语** — "在当今社会""众所周知"→精简
21. **过度限定** — "在某种程度上""从某个角度来说"→直接陈述
22. **空洞乐观结尾** — "未来可期""让我们拭目以待"→用具体计划替代或删除
23. **禁止表达** — 总而言之、值得注意的是、首先其次最后→删除或改写
24. **禁止模式** — 清单体、过度解释、制造金句、喊口号→改为自然叙述

## 评分体系

### 爆款 6 维度（每项 0-10 分）

| 维度       | 评判标准                               |
| ---------- | -------------------------------------- |
| 好奇心缺口 | 是否制造了点击冲动？                   |
| 情绪共鸣   | 是否触发高唤醒情绪（愤怒/惊讶/感动）？ |
| 价值实用性 | 是否值得收藏转发？                     |
| 关联时效性 | 为什么现在要看？                       |
| 叙事节奏   | 是否有滑梯效应（读了开头停不下来）？   |
| 反直觉新颖 | 是否挑战了读者的既有认知？             |

### 人味 5 维度（每项 0-10 分）

| 维度   | 评判标准                 |
| ------ | ------------------------ |
| 直接性 | 直接陈述 vs 绕圈宣告     |
| 节奏   | 长短交错 vs 机械重复     |
| 信任度 | 简洁信任读者 vs 过度解释 |
| 真实性 | 自然口语感 vs 生硬书面腔 |
| 精炼度 | 无冗余 vs 废话连篇       |

## 输出格式

```
## 优化后的文本

[完整的优化后文本]

## 评分

**爆款潜力**：XX/60（好奇心X + 情绪X + 价值X + 关联X + 节奏X + 新颖X）
**人味评分**：XX/50（直接X + 节奏X + 信任X + 真实X + 精炼X）

## 检测到的 AI 模式

- 模式7: AI高频词汇 — "赋能"→"帮助"，"格局"→删除
- 模式17: 协作痕迹 — 删除"希望对您有帮助"
- ...

## 主要改动说明

1. ...
2. ...
```

## 注意事项

- **语言**：所有输出使用简体中文（专有名词除外，如 AI、Twitter、iPhone）
- **如果用户提了具体修改意见**：优先体现用户意见，再做 AI 模式检测
- **不要过度改写**：保留原文的核心观点和信息，只去除 AI 痕迹
- **保持原文风格**：如果原文是口语化风格，优化后也应保持口语化
